# Character_Festival
천하제일 캐릭터 경진대회🍀
- - -
## 뽀길이(bbogil)
뽀(뽀글) + 길이(바다코끼리)에서 영감을 받았습니다.
10초정도 지켜봐주세요!
- - -
### 기능 설명
* 입을 벌리면서 물풍선을 충전한다.
* 물풍선이 회전한다.
* 물풍선이 훨씬 커지다가 터진다.
* 이름이 나타난다.
- - -
### 참고
* '뽀길이' 헤딩요소의 글꼴('font-family')은 네이버에서 무료로 배포하는 '나눔손글씨 배은혜체'입니다.
* 머리는 svg 파일을 사용했고, path 태그 내에 style 속성을 지정해주면 색깔을 변경할 수 있습니다.
* HTML Validator✅
